Descrição

O Stats4Good é um grupo de estudantes entusiastas da ciência dos dados criado em 2016. Nosso objetivo é fornecer estatísticas e análise de dados como ferramentas de auxilio à sociedade. Além disso, os pilares do grupo são: Ensinar, Aprender e Discutir.

Objetivos

O objetivo principal é a utilização de dados livres para prover informações de valor para a sociedade. Muitas são as fontes de dados disponíveis no Brasil. Porém, nem toda a população possui o conhecimento necessário para obter tais dados de forma simples. Além disso o processo de limpeza, manipulação e geração de informações não é uma tarefa trivial. Desta forma, o Stats4Good através de profissionais capacitados atua na geração de informações para a população a respeito de assuntos relevantes. Hoje (16/03/2016) o grupo trabalha em dois projetos: (1) criação de uma ferramente de visualização dos dados do portal da transparência (http://www.portaltransparencia.gov.br); (2) criação de uma plataforma cujo objetivo é descrever o problema do estupro em mulheres no Brasil.

Tecnologias

Diversas são as tecnólogias utilizadas visto que os projetos atuam desde a extração dos dados até a visualização de dados. No blog do grupo alguns tutoriais podem ser encontrados (https://stats4good.github.io/blog/).

Período de atuação

De Março de 2016 até o presente.
